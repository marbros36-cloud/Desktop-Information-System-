# Desktop Information System

A desktop application developed in C# as part of my Computing degree. The system was designed to manage student, teacher, and administrative information within an educational environment.

## Features

- Student record management
- Teacher management
- Administrative record management
- Add, edit, delete, and search functionality
- Database integration
- User-friendly interface

## Technologies Used

- C#
- .NET
- SQL Database
- Visual Studio

## Skills Demonstrated

- Object-oriented programming
- Database design
- CRUD operations
- Software development lifecycle
- Problem solving


## How to Run

1. Open the solution in Visual Studio.
2. Restore any required packages.
3. Configure the database connection if necessary.
4. Build and run the application.
